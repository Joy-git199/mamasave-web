MamaSave

Smart Maternal Health Monitoring System

About the Project

MamaSave is a wearable health monitoring belt designed for pregnant women to track vital signs such as heart rate, temperature, and SpO2. It sends emergency alerts via GSM and GPS and stores data to Firebase, which is accessible to caregivers and health workers through a Flutter app.

System Features

  • Real-time heart rate, SpO2, and temperature monitoring
  • Automatic emergency alerts via SIM800L
  • GPS tracking using NEO-6M
  • Cloud storage via Firebase
  • Flutter mobile app for caregivers
